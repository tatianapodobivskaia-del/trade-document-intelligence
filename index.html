Cyrillic Transliteration Engine

Unique Feature

Russian entity names can appear in 3-5 different Latin spellings across trade documents. Standard fuzzy matching misses these variations. This engine generates multiple transliteration variants using ISO 9, BGN/PCGN, passport, and informal systems.
